プログラムの種類は何でも大丈夫です。

https://claude.com/ja/download 

ログインは開発者に聞いてください。

Gemini 設置

https://antigravity.google/

GPT 設置

 https://openai.com/ja-JP/index/introducing-the-codex-app/

サンプールファイル

下のリンクからファイルをダウンロードします。

https://github.com/midasit-developer-01/plug-in-template.git

設定

plug-in-template.zip のファイルを開けてinstall-git.batファイルをダブルクリックします。

install-git.bat は Gitというプログラムを設置します。

プログラム連絡

Claude

Codeのどころで+New Sessionをクリック。

下のフォルダの選択でダウンロードしたplug-in-templateを指定します。

Gemini

File をクリック、Create Projectをクリックします。

Add Folderのどころでダウンロードしたplug-in-templateを指定します。



GPT(== Codex)

New Chat(New Session)をクリックします。

下のプロジェクトで作業をクリック、既存のフォルダ選択でダウンロードしたplug-in-templateを指定します。

たまに、下のようなCodexがほかのAgent。。。何とか。。。設定を探しましたと出た場合、適当に黒いバートンをクリックします。

共通作業

完成した作業は、AI にビルド(build)するよう依頼します。

該当の build ファイルを圧縮します。

注意！：build フォルダ内部のデータのみを圧縮！！！

Civil / Gen で Plug In を開き、My Work カテゴリを選択した後、アップロード（雲アイコン）を選択します。

タイプ（new / update）、タイトル、説明をそれぞれ入力した後、Select Zip File を選択して圧縮したファイルをアップロードします。

Run ボタンを押すと、作業した Plug In の実行を確認できます。

UI のサイズ調整やタイトル（Title）の修正をしたい場合は、manifest.json ファイルを調整するよう AI に依頼してください。
